## Content Structure Patterns

### Timeline/Process
```
Use numbered boxes connected by arrows
Box 1 → Box 2 → Box 3 → Box 4
Each box contains:
- Step number
- Title
- 2-3 bullet points
- Optional icon
```

### Comparison
```
Split layout with two columns
Left side: Topic A (color 1)
Right side: Topic B (color 2)
VS in center
Each side has 3-4 key points
```

### Hierarchy/List
```
Main topic at top
Subtopics as colored boxes
Details as bullet points under each
Use nesting for depth
```

### Formula/Equation
```
Central equation in large text
Components explained around it
Arrows pointing to each part
Real-world example at bottom
```

### Tips/Best Practices
```
Title at top
Grid of tip cards
Each card: icon + title + one sentence
Use ✅ for "do" and ❌ for "don't"
```

## Brand Element Formats

### Header Format
```
=== HEADER TOP ===
Top left: Large bold "[BRAND NAME]"
Below: Small subtitle "[TAGLINE]"
Title: "[IMAGE TITLE]"
Add doodle decorations (dots, stars, squiggles)
```

### Footer Format
```
=== FOOTER BOTTOM ===
Horizontal banner: "[BRAND] | [TAGLINE]"
Small text: "Made with [BRAND]"
Add doodle decorations (hearts, stars)
```
